# WebAuthn Demo

A demonstration of passwordless authentication using WebAuthn API with Node.js backend and React frontend.

## Features

- Passwordless authentication via biometrics or security keys
- Secure registration and authentication flow
- React frontend with modern UI
- Node.js/Express backend with WebAuthn integration

## Prerequisites

- Node.js v16 or higher
- npm v8 or higher
- Modern browser with WebAuthn support (Chrome, Firefox, Edge, Safari)
- Optional: Physical security key (YubiKey, Titan, etc.) or biometric authenticator

   cd ./api

2. Run backend & client: 
 ```sh
